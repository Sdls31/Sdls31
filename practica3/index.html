Curso Attiny85 básico.

José Sebastián De los Santos Hernández
Ing. Sistemas computacionales
Fecha: Septiembre 22, 2023
Arduino Electrónica Attiny85

- Resumen -

La práctica consistió en desarrollar 14 prácticas, utilizando por primera vez un microcontrolador que en este caso será el Attiny85, antes de empezar a desarrollar todos los circuitos, se tenia que hacer un procedimiento, que consistia en hacer un burn boatloader y programar el arduino "as isp" para poder usarlo y poder programarlo sin ningún problema.

- Introducción -

Durante esta práctica se utilizaron distintos materiales y se siguió con la placa Arduino UNO. Pero esta vez además de usar el atmega que incluye nuestra placa, utilizaremos el attiny que tiene 8 puertos y que es muy sencillo de utilizar. Durante toda la práctica se prenderan leds y se utilizaran botones para prender leds.

Podemos hacer prácticas similares a la 1 y 2, pero esta vez con el Attiny85.

- Materiales -

Se utilizaron pocos materiales, incluso muy similares a la práctica #1.

  • Leds de 5v
  • Protoboards
  • Resistencias de 220Ω
  • Resistencias de 10kΩ
  • Botones de 2 pines
  • Jumpers Macho-Macho
  • Placa Arduino UNO
  • Microcontrolador Attiny85

- Desarrollo -

La primer práctica, en este caso la 0; se tratá de las conexiones que se tienen que hacer para el burn boatloader. Es importante recordar que el codigo se debe cargar en la primera configuración para despues cambiar de lugar el Attiny85. La práctica #1, utilizariamos un led para hacerlo parpadear con ayuda del Attiny85. La práctica #2, mantendría el mismo circuito pero esta vez lo mantendremos prendido. Pasando por la práctica #3, que tendrá el mismo circuito que las 2 anteriores, pero el led estará apagado. La práctica #4, se encargaria de hacer papadear al led mas rapido. La práctica 5, incorporaremos 1 leds más y ambos haran un parpadeo pero lo haran de manera alternante. Para la siguiente daremos un salgo a poner 3 leds los cuales haran un contador.

Para la práctica #7, tendremos un boton y un led, donde al presionar el boton prenderá el led. La práctica #8 es lo mismo que la 7 pero con un código distinto. En la práctica #9, hará lo mismo que la 7 y la 8 pero con un código distinto. Enseguida tenemos la práctica #10, en la que tendremos 2 botones para simular la compuerta lógica OR. Para la práctica #11, es el mismo circuito que la 10 pero esta vez simularemos la compuerta lógica AND. Enseguida la práctica #12, que consta de un contador aprentando un boton y con 3 leds. Para la ultima práctica, tenemos un contador con 2 botones, en la que un boton aumenta y el otro disminuye este contador.

Simulación

A continuación se mostrarán las simulaciones de las práctica en tinkercad.

Diseño

Se mostrarán los códigos que hacen funcionar nuestro microcontrolador y por tal nuestro circuito.

- Resultados -

Los resultados fueron positivos aunque se presentaron unos errores para hacer que el arduino funcionara como isp, pero fuera de ello las práctica se culminó en un buen modo.

- Conclusiones -

Podemos concluir, que el microcontrolador en un componente realmente importante dentro de la electrónica, ya no solo refiriendonos al Attiny85, por que recordemos que arduino tiene el ATMEGA328P que igual nos ayuda demasiado, se pueden rescatar muchas cosas como las características de nuestros microcontroladores y es que ambos son de 8 bits, 8 veces menos que nuestra computadora tal y como la conocemos en la actualidad.
